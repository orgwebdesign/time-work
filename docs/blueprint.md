# **App Name**: TaskMaster

## Core Features:

- Task Management: Create, read, update, and delete tasks in the to-do list.
- Lists: Categorize tasks into lists to enable users to manage many different goals (work, home, family, etc).
- Due Dates and Reminders: Set due dates and reminders for tasks, so you don't forget the important things.
- Smart Task Suggestions: Suggest similar, relevant, or prerequisite tasks, to give you ideas and keep you organized; the tool decides when and if to suggest tasks.
- Completion and Archiving: Mark tasks as complete to keep the list current, and optionally move them to an archive.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6) to promote focus and productivity.
- Background color: Light gray (#EEEEEE), for a clean, distraction-free backdrop.
- Accent color: Soft green (#A5D6A7) for completed tasks and positive feedback.
- Body and headline font: 'Inter', a grotesque-style sans-serif font that looks modern and neutral.
- Simple, outlined icons to represent task categories and actions.
- Clean and spacious layout with clear visual hierarchy.
- Subtle animations to confirm actions and provide gentle feedback.
